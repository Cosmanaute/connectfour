# Connectfour in rust
 
 Made for linux. 
 May have problems 
 working on windows.
 
## How to play

The game consists of two players: O and X.   
To choose which column to move to,  
you just input a number between 1 and 7.    
1 is far left and 7 is far right.    
      
4 connecting pieces including diagonals == win.
